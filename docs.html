Database API Documentation

SQLite Database with Users and Posts

Base URL: http://localhost:8000

Available Endpoints

GET /

Welcome message and API information

GET /users

Get all users from the database

POST /users

Create a new user

Request Body: { "username": "testuser", "email": "test@example.com", "full_name": "Test User" }
GET /posts

Get all posts with author information

POST /posts

Create a new post

Request Body: { "title": "My First Post", "content": "Hello World!", "author_id": 1 }

Database Schema

Users Table

  • id: INTEGER PRIMARY KEY
  • username: TEXT UNIQUE NOT NULL
  • email: TEXT UNIQUE NOT NULL
  • full_name: TEXT (optional)
  • is_active: BOOLEAN DEFAULT 1
  • created_at: TIMESTAMP DEFAULT CURRENT_TIMESTAMP
  • updated_at: TIMESTAMP DEFAULT CURRENT_TIMESTAMP

Posts Table

  • id: INTEGER PRIMARY KEY
  • title: TEXT NOT NULL
  • content: TEXT (optional)
  • is_published: BOOLEAN DEFAULT 0
  • created_at: TIMESTAMP DEFAULT CURRENT_TIMESTAMP
  • updated_at: TIMESTAMP DEFAULT CURRENT_TIMESTAMP
  • author_id: INTEGER (Foreign Key to users.id)

cURL Examples

# Get all users curl -X GET http://localhost:8000/users # Create a user curl -X POST http://localhost:8000/users \ -H "Content-Type: application/json" \ -d '{"username": "testuser", "email": "test@example.com", "full_name": "Test User"}' # Get all posts curl -X GET http://localhost:8000/posts # Create a post curl -X POST http://localhost:8000/posts \ -H "Content-Type: application/json" \ -d '{"title": "My First Post", "content": "Hello World!", "author_id": 1}'
